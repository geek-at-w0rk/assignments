# Babylon.js Extrude Face Application

This application demonstrates a simple extrusion effect on a cube's faces using the Babylon.js framework. The user can interact with the cube to select a face and then perform an extrusion-like effect by clicking and dragging the mouse.

## Getting Started

To run the application locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the `index.html` file in a modern web browser (e.g., Google Chrome).

## Usage

- Click on a face of the cube to select it for extrusion.
- While holding down the mouse button, move the mouse vertically to perform the extrusion-like effect.
- Release the mouse button to stop the extrusion.

## Dependencies

- [Babylon.js](https://www.babylonjs.com/): A powerful 3D engine for rendering graphics in the browser.
